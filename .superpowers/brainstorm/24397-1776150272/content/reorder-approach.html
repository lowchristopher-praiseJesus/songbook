How should songs be reordered within a collection?

Three approaches — each with a different interaction model

⠿ El Shaddai
⠿ Blessed Be Your Name

A — Drag handles (always visible)

Every song has a drag handle (⠿) on the left. Drag to reorder anytime. Simple and discoverable.

Pros

  • Immediate, no mode switch
  • Familiar pattern

Cons

  • Handles clutter normal use
  • Requires drag-and-drop library

El Shaddai ↑ ↓
Blessed Be Your Name ↑ ↓

B — Up/Down arrow buttons

Each song row shows ↑↓ buttons. Click to move one position. No extra library needed.

Pros

  • No drag-and-drop library
  • Works well on mobile/touch
  • Accessible by default

Cons

  • Slow for large reorders
  • Arrows add visual noise

C — Reorder mode (toggle)

A small "Reorder" link in the collection header activates drag-and-drop. Normal view stays clean. "Done" exits the mode.

Pros

  • Cleanest normal view
  • Intent is explicit

Cons

  • Extra step to reorder
  • Still needs drag library
