Key behaviours

  • Segmented control sits between the search box and the song list
  • All Songs is sorted A–Z by title with letter dividers; shows title + artist
  • Prev/next navigation (swipe/arrows) follows the active view's order
  • Search works the same in both views — filters the active list
  • Export mode works the same — checkboxes on individual songs; no letter-group select in All Songs (letters aren't meaningful collections)
  • Preference persisted to localStorage (songsheet_view_mode), restored on load
