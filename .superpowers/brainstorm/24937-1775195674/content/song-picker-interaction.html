How should songs be added to a collection?

Once a collection is created, how does the user pick which songs go in it?

A

Modal Picker

Each collection has an "Add Songs" button. Clicking it opens a modal showing all songs in the library — user checks the ones to include, then confirms.

✅ Sunday Set
B

Right-click / Long-press on Song

In "All Songs" view, right-clicking (or long-pressing on mobile) any song shows a context menu: "Add to Collection →" with a submenu of existing collections to choose from.

C

Inline "Add Songs" inside collection

When a collection is expanded in the sidebar, there's a small "+ Add songs" row at the bottom. Clicking it opens an inline mini-search/list to pick songs without a modal.
