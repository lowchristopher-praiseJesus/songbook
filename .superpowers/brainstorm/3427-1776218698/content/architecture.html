API Endpoints

  • POST /session/create — new session, returns leader + member tokens
  • GET  /session/:code/state?since=N — poll for ops since seq N
  • POST /session/:code/op — append an operation
  • POST /session/:code/lock/:songId — acquire edit lock
  • DELETE /session/:code/lock/:songId — release edit lock
  • POST /session/:code/close — leader closes session (requires leader token)
