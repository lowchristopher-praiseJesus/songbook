How to Get a Firecrawl API Key

A quick, step-by-step guide to signing up and grabbing your API key in under two minutes.

1

Go to the Firecrawl Homepage

Open your browser and navigate to firecrawl.dev. You'll see the main landing page with the tagline "Power AI agents with clean web data". Look for the orange Sign up button in the top-right corner of the navigation bar and click it.

The Firecrawl homepage — click the orange "Sign up" button in the top-right corner.
2

Create Your Account

After clicking Sign up, you'll be taken to the account creation page. Make sure the Sign Up tab is selected at the top. You have three options to create your account:

  • 📧 Email & Password — Enter your email and choose a password, then click the orange Create Account button.
  • G Continue with Google — Sign in instantly using your Google account. Fastest option if you're already logged into Google.
  • 🐙 Continue with GitHub — Ideal for developers who prefer to link their GitHub identity.
Tip: Choose whichever method is most convenient. All three options give you the same access and free-tier credits.
The Sign Up page — use email/password, Google, or GitHub to create your account.
3

Copy Your API Key from the Dashboard

Once you've signed up (or logged in), you'll land on the Firecrawl Dashboard. Your API key is displayed right on the Overview page in the "API Key" section on the right side.

The key is partially masked for security (e.g. fc-2•••••••28f9). To copy it:

  • 👁️ Click the eye icon to reveal the full key.
  • 📋 Click the copy icon (next to the eye) to copy the key to your clipboard.
Keep it safe: Treat your API key like a password. Don't commit it to public repositories or share it publicly. Use environment variables to store it in your projects.
The Dashboard — your API key is in the circled section on the right. Click the copy icon to grab it.
✓

You're All Set!

Paste your API key into your project's configuration or environment variables and start using the Firecrawl API to search, scrape, and crawl the web. Check out the Firecrawl Docs for integration guides and code examples.
